Select DNS Providers

Choose only the DNS providers you want for your DoH Well Known Servers. The registry file will include only your selections.

After downloading: Import the .reg file into Windows, reboot your system, then set your desired DNS provider in Windows network settings.

For inspiration, check out the HTML source of this page and my prior Extended DoH Well Known Servers release. This project was made to empower users to choose their own DNS providers.

Why? To make secure DNS configuration easy, flexible, and accessible for everyone.
